## Installation

Clone the repo:

```

git clone --depth https://github.com/viveks77/twitter-clone.git

```
Install the dependencies:

```

yarn

```

## Commands 

Running locally :  ``yarn run dev``

Open http://localhost:3000 with your browser to see the result.
